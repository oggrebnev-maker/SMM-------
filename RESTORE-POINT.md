# Точка восстановления — шаблоны (хэштеги, подпись)

**Дата:** 2025-03-11

## Что сделано в этой сессии

### Вкладка «Хэштеги» (посты и комментарии)
- Режимы «добавление» / «просмотр»: при наличии хэштегов форма скрыта, показываются кнопка «Редактировать» и список тегов.
- Список тегов: каждый тег в виде сервисной кнопки с `:: #тег`, красный круг с крестом для удаления.
- Одна нижняя кнопка «Сохранить»/«Редактировать»: при наличии тегов и закрытой форме — «Редактировать» (открывает форму), при открытой форме — «Сохранить».
- Drag & Drop для изменения порядка тегов с сохранением на сервер.
- Подзаголовок «Добавленные хэштеги» в стиле формы.

### Вкладка «Подпись»
- Отображение текущей подписи: при сохранённой подписи показывается блок «Текущая подпись» (название, использование, текст).
- Та же логика кнопки: «Редактировать» открывает форму, «Сохранить» сохраняет и возвращает к просмотру.

### Затронутые файлы
- `assets/js/pages/projects.js` — разметка панелей, `updateProjectViewHashtagsPostPanel`, `updateProjectViewHashtagsCommentPanel`, `updateProjectViewSignaturePanel`, `updateHashtagsMainButton`, drag/drop, сохранение.
- `assets/css/app.css` — стили списка тегов, кнопки удаления, раскрывающегося блока, отображения подписи.
